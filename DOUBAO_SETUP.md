# 豆包API配置说明

## 概述

本项目已集成豆包大模型（火山引擎）用于识别基金持仓截图中的基金信息。

## 配置步骤

### 1. 获取火山引擎API密钥

1. 访问 [火山引擎控制台](https://console.volcengine.com)
2. 注册/登录账号
3. 进入 [API访问密钥管理](https://console.volcengine.com/iam/keymanage)
4. 创建新的访问密钥（Access Key ID 和 Secret Access Key）

### 2. 创建豆包接入点

1. 进入 [火山引擎方舟](https://console.volcengine.com/ark)
2. 在左侧菜单选择「接入点管理」
3. 点击「创建接入点」
4. 选择模型（建议：doubao-pro-vision 或 doubao-lite-vision）
5. 填写接入点名称（如：fund-portfolio-eval）
6. 点击「确定」创建
7. 复制生成的接入点ID（格式如：ep-20241212121212-xxxxx）

### 3. 配置环境变量

在项目目录下创建 `.env` 文件（如果还没有）：

```bash
# 豆包API配置
DOUBAO_API_KEY=你的API密钥
DOUBAO_ENDPOINT=https://ark.cn-beijing.volces.com/api/v3/chat/completions
DOUBAO_MODEL=你的接入点ID
```

或者直接在启动应用前设置环境变量：

```bash
export DOUBAO_API_KEY="你的API密钥"
export DOUBAO_MODEL="ep-20241212121212-xxxxx"
```

### 4. 重启应用

重启你的Flask应用以使配置生效。

## 使用说明

### 访问页面

启动应用后，访问以下页面：

```
http://localhost:5001/portfolio-eval
```

### 使用流程

1. **上传截图**：点击上传区域，选择你的基金持仓截图（支持天天基金、支付宝、腾讯理财通等平台）
2. **手动输入**：或者点击下方链接，手动输入基金代码和占比
3. **分析组合**：点击「开始分析」按钮
4. **查看结果**：
   - 组合整体指标
   - 持仓明细
   - 风格分析
   - 行业分布
   - 历史回测（可选1年/2年/3年）
   - 调仓建议

### 示例数据

如果API未配置，系统会使用示例数据进行演示，包含以下基金：

- 161725 招商中证白酒指数（30%）
- 005918 易方达蓝筹精选混合（25%）
- 163406 兴全合润混合（25%）
- 519674 银河创新成长混合（20%）

## 模型选择

豆包提供多种视觉模型供选择：

| 模型 | 推荐场景 | 特点 |
|------|---------|------|
| doubao-pro-vision | 生产环境 | 识别准确率高，支持复杂场景 |
| doubao-lite-vision | 测试/低流量 | 响应快，成本低 |

## 常见问题

### Q: 识别准确率不高怎么办？

A: 
1. 确保截图清晰，包含完整的基金代码、名称和金额信息
2. 尝试使用 `doubao-pro-vision` 模型获得更好的识别效果
3. 可以手动调整识别结果

### Q: 可以支持哪些平台的截图？

A: 
- 天天基金
- 支付宝
- 腾讯理财通
- 其他主流基金平台

### Q: API调用失败怎么办？

A: 
1. 检查API密钥和接入点ID是否正确
2. 确认账户有足够的余额
3. 查看控制台错误日志
4. 如果未配置API，系统会自动使用示例数据

### Q: 图片大小有限制吗？

A: 建议图片大小不超过10MB，分辨率在2000x1500以内。

## 安全提示

- 不要将包含API密钥的 `.env` 文件提交到代码仓库
- 定期轮换API密钥
- 监控API调用量和费用
- 不要在前端暴露API密钥

## 技术支持

如果遇到问题：

1. 查看应用日志
2. 检查火山引擎控制台的API调用记录
3. 确认网络连接正常
4. 验证模型接入点是否处于运行状态
